Notebook Examples
=================

The project includes interactive notebooks to demonstrate the simulator behavior.

Bell State Walkthrough
----------------------

This notebook builds a Bell circuit, runs your custom simulator, compares against a Qiskit reference statevector, and visualizes measurement probabilities.

How to run locally
------------------

1. Start Jupyter Lab from the project root.
2. Open the notebook at docs/notebooks/bell_demo.ipynb.
3. Open docs/notebooks/simulator_quickstart.ipynb for the quickstart workflow.
4. Open docs/notebooks/statevector_benchmark_comparison.ipynb for benchmark plots and speedup analysis.
5. Run all cells.
